В конец документа

Данное сообщение появляется, если вы распаковываете по умолчанию(Стоит галочка архивы в рабочем каталоге), но архива по умолчанию не создавался, дальнейшие действия:

  1. Нажмите кнопку "ОК. Для выхода в подменю Архивы баз данных.
  2. Скопируйте файл archive.rar в папку - "Archive:"
    • Найдите файл "Archive.rar"
      • Зайдите в диск С и введите в строку поиск следующее - "archive.rar", смотри рисунок 1.

        Рисунок 1 - Поиск файла archive.rar

    • Скопируйте файл archive.rar в папку "Archive", в корне программы
      • Нажмите правой кнопкой мыши на файле archive.rar.
      • В появившемся подменю нажмите копировать.
      • Пройдите в папку программы(При установке по умолчанию программа ставитcя в C:\Program Files (x86)).
      • Зайдите в папку Archive.
      • Нажмите правой кнопкой мыши на свободном месте.
      • В появившемся подменю выберите вставить.
  3. Укажите путь куда последний раз делали архив.
  4. Зайдите в программу АСПУ, в подсистему архивы баз данных и извлеките архив.

Примечание:

  1. Копировать - комбинация клавиш - Ctrl+C.
  2. Вставить - Ctrl+V.
  3. Пример пути - "C:\Program Files".
