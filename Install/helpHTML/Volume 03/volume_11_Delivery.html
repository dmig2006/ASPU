Подсистема "Выдача"

В конец документа

В подсистеме "Выдача" предоставлена возможность учета выданных элементов ЗИП. Внешний вид окна представлен на рисунке 1.


Рисунок 1 - Подсистема выдача

Назначение клавиш:

  1. "Об элементе" - просмотр полной информации об элементе.
  2. "Поиск" - поиск необходимого элемента по атрибутам. При нажатии на этой кнопки станет активным окно поиска смотри п.3.12 (Подсистема "Поиск").
  3. "Накладная" - вывод накладной по выданным элементам(становится доступной после выдачи элементов).
  4. "Инф.листок" - вывод отчета по требуемым элементам(становится доступной после выдачи элементов).
  5. "Начать регистрацию" - включение функции регистрации штрих-кодов с помощью терминала или вручную.
  6. "Помощь" - позволяет открыть справку.
  7. "Выход" - осуществляет выход из окна подсистемы "Выдача".

Клавиши управления перечнем требуемых элементов:

  1. - копировать активный элемент в перечень требуемых элементов.
  2. - удалить активный элемент из перечня требуемых элементов.
  3. - удалить все элементы из перечня требуемых элементов.
  4. - для перехода к последней надписи.
  5. - для перехода к предыдущей надписи.
  6. - для перехода к следующей надписи.
  7. - для перехода к первой надписи.

Назначение полей следующее:

  • "Обозначение" - Обозначение элемента.
  • "Наименование" - Имя элемента.
  • "Кол" - Количество полученных элементов.

Для выдачи элементов в системе АСПУ необходимо выполнить следующие действия:

  1. Из перечня элементов, хранимых в машине, выбрать и занести в перечень требуемых необходимые элементы ЗИП с помощью клавиши управления, смотри рисунок 2.

    Рисунок 2 - Количество затребовших элементов
  2. Нажать клавишу "Начать регистрацию" для включения функции регистрация штрих-кодов с помощью терминала.
  3. Курсор (мигающая вертикальная линия) должен находится в появившемся окне ввода, где будут отражаться считываемые терминалом штрих-коды, иначе коды элементов не будут зарегистрированы.
  4. Считывание терминалом:
    • Подвести терминал к штрих-коду.
    Ввод с клавиатуры:
    • Ввести штрих код в поле.
    • Нажать на клавиатуре клавишу "Enter"
    Если количество элементов одного типа больше одного, вводить через терминал или вручную надо на каждый элемент.
  5. Нажать клавишу "Сохранить" для записи информации по выданным элементам в систему АСПУ, при этом надо будет ввести данные:
    • ФИО экспедитора, смотри рисунок 3.

      Рисунок 3 - Экспедитор
    • ФИО получателя, смотри рисунок 4.

      Рисунок 4 - Получатель
  6. Становится доступной клавиша "Накладная" для вывода на печать документа о выданных элементах, смотри рисунок 5.

    Рисунок 5 - Накладная
  7. Для вывода на печать спсика выданных элементов, нужно нажать клавишу "Инф.листок", смотри рисунок 6.

    Рисунок 6 - Инф.листок
  8. При необходимости повтора операции выдачи, необходимо выйти в главную форму, и войти в подсистему "Выдача" повторно.
