Подсистема "Пользователи"

В конец документа

Данная подсистема предназначена для регистрации пользователей системы и доступна только администраторам (пользователям с полным правом доступа к данным). Она активизируется путем нажатия кнопки "Пользователи" главной формы АСПУ.

При входе в подсистему на экран выводится окно со списком учетных записей пользователей зарегестрированных в системе, внешний вид формы представлен на рисунке 1


Рисунок 1 - Таблица пользователей

В таблице отображены учетные записи пользователей: регистрационное имя, фамилия имя отчество, звание. Активная учетная запись подсвечивается. Администратору предоставляются следующие возможности:

  1. Переход к последней надписи, надо нажать кнопку - .
  2. Переход к предыдущей надписи, надо нажать кнопку - .
  3. Переход к следующей надписи, надо нажать кнопку - .
  4. Перехода к первой надписи, надо нажать кнопку - .
  5. Удаление учетной записи пользователя, надо нажать кнопку "Удалить".
  6. Регистрация нового пользователя системы, надо нажать кнопку "Добавить".
  7. Корректирование информации о пользователе,надо нажать кнопку "Изменить".
  8. Для вызова справки надо нажать кнопку "Помощь".
  9. Для выхода из режима администрирования системы, надо нажать кнопку "Выход".

Под регистрацией нового пользователя системы подразумевается добавление новой учетной записи. При нажатии кнопки "Добавить". На экране откроется окно регистрации пользователя, внешний вид которого представлен на рисунке 2


Рисунок 2 - Регистрация пользователя

Для регистрации пользователя необходимо:

  • в поле "Имя" ввести регистрационное имя пользователя.
  • в поля "Пароль" и "Повтор" ввести пароль регистрируемого пользователя.
  • в поле "Фамилия, Имя, Отчество" ввести фамилию, имя и отчество регистрируемого пользователя.
  • в поле "Звание" ввести его звание.

Вводить можно любые символы.

Для предоставления пользователю прав администратора необходимо поставить галочку под надписью "Право полного доступа к данным".

Для добавления новой учетной записи нажмите кнопку "Ок" или "Отмена" для отмены регистрации. Форма регистрации скроется и на экране вновь отобразится окно со списком учетных записей зарегестрированных в системе пользователей.

Для корректирования информации о пользователе необходимо выделить необходимую учетную запись в предоставленном списке и нажать кнопку "Изменить". Откроется окно изменения учетной записи пользователя, внешний вид которого предоставлен на рисунке 3.

Для изменения информации о пользователе необходимо внести новые данные в те поля, значение которых надо изменить. Для подтверждения изменений в учетной записи нажмите кнопку "Ок", для отмены изменений - кнопку "Отмена". Окно изменения учетной записи пользователя скроется и вновь отобразится окно со списком учетных записей зарегестрированных в системе пользователей.


Рисунок 3 - Изменение информации о пользователе

При выход из под системы "Пользователи", окно со списком учетных записей пользователей скроется и на экране вновь отобразится главная форма системы.
