Подсистема "ЗИП"

В конец документа

Данный режим предоставляет возможность просмотра полной информации об элементах ЗИП-Г. Режим активизируется путем нажатия кнопки "ЗИП" главное формы АСПУ. При входе в подсистему просмотра базы данных на экране отоброзится окно "Элементы группового комплекта ЗИП", внешний вид которого представлен на рисунке 1.


Рисунок 1 - Элементы ЗИП

В представленном окне отображается полная информация о текущем элементе ЗИП-Г

Назначение полей:

  1. "Наименование" - полное наименование элемента.
  2. "Штрих-код" - штриховой код элемента.
  3. "Обозначение" - обозначение элемента.
  4. "Комплект" - комплект, к которому он принадлежит.
  5. "Местоположение" - расположение элемента в формате С.С.Т.ЩЩЩ.ЯЯЯ. где:
    • С - номер склада.
    • С - номер стеллажа (шкаф).
    • Т - номер тумбочки.
    • ЩЩЩ - номер ящика.
    • ЯЯЯ - номер ячейки.
  6. "Склад" - наименование склада, где находится элемент ЗИП.
  7. "Габариты" - габаритные размеры элемента.
  8. "Дата изготовления" - дата изготовления элемента.
  9. "Срок годности(в месяцах)" - гарантийный срок использования элемента в месяцах.
  10. "Количество(имеющееся)" - количество элементов, имеющееся в наличии.
  11. "Количество(максимальное)" - максимальное количество элементов, которое возможно разместить в машине.
  12. "Единицы измерения" - информация о единицах измерения (шт., м., кг.), которым измеряется количество элементов.
  13. "Примечание" - дополнительная информация об элементе (в том числе информация об особых условиях хранения).

Назначение кнопок:

  1. - Для перехода к последнему элементу
  2. - Для перехода к предыдущему элементу
  3. - Для перехода к следующему элементу
  4. - Для перехода к первому элементу
  5. Кнопка "Отчет". При нажатии будет открыто окно для печати документа, печатная форма которого представлена на рисунке 2.


    Рисунок 2 - Окно печати документа
  6. Кнопка "Поиск".При нажатии открывается окно подсистемы "Поиск", которая предоставляется возможность быстрого перемещения к искомому элементу. Порядок работы описан в п.3.12(Подсистема поиск)
  7. Кнопка "Помощь". При нажатии откроется справка.
  8. Кнопка "Выход"". При нажатии осуществляется выход из под системы ЗИП.
