Вид главной формы: смотри рисунок 1


Рисунок 1-Главное меню.

Пользователю предоставляется возможность работать в следующих режимах:

  1. Поиск элементов ЗИП-Г. Вызывается нажатием кнопки "Поиск". Предназначен для поиска и быстрого перехода к необходимому элементу.
  2. Выдача элементов ЗИП-Г. Вызывается нажатием кнопки "Выдача". Предназначен для автоматизации операции выдачи элементов.
  3. Прием элементов ЗИП-Г. Вызывается нажатием кнопки "Прием". Предназначен для автоматизации операции приема элементов.
  4. Вывод перечня элементов, требующих пополнение или замены. Вызывается нажатием кнопки "Требуется". Предназначен для просмотра и вывода на печать списка элементов, запасы которых необходимо пополнить (нет в наличии на складе) или заменить (истек срок годности).
  5. Корректировка базы данных. Вызывается нажатием кнопки "Справочник". Предназначен для изменения и добавления элементов ЗИП-Г (Только для администратора).
  6. Ведение справочника штрих-кодов ящиков (панелей) с просмотром спиcка содержимого ящика (панели) вызывается нажатием кнопки "Коды ящиков"
  7. Администрирование системы. Вызывается путем нажатия кнопки "Пользователи". Предназначен для регистрации пользователей системы (только администратор).
  8. Просмотр базы данных. Вызывается нажатием кнопки "ЗИП". Предназначен для просмотра подробной информации об элементах ЗИП.
  9. Проведение ревизии. Вызывается нажатием кнопки "Ревизия". Предназначен для проверки наличия всех необходимых элементов и, при необходимости, изменения их фактического количества.
  10. Вызов справки системы, вызывается кнопкой "Помощь".
  11. Ведение архивов баз данных. Для хранения и аварийного восстановления информации, вызывается нажатием кнопки "Архивы БД".
  12. Индексирование текущего состояния базы данных. Для аварийного восстановления индексов, к текущим базам данных, вызывается нажатием кнопки "Индексы БД".
  13. Экспортирование и импортирование информации по складу. Данная функция необходима для хранения информации по складу и передачи информации от одного локального компьютера к другому. Вызывается нажатием кнопки "Склады."
  14. Выход из системы осуществляется нажатием кнопки "Выход".
  15. Переключение языка осуществляется следующим образом:
    • Изначально стоит русский, смотри рисунок 2:


      Рисунок 2 - Язык системы русский.

      Интерфейс будет на французском, если выбрать - "французский", на русском если выбрать - "русский".
    • Изначально интерфейс был на французском, смотри рисунок 3:


      Рисунок 3 - язык системы французский.

      Интерфейс будет на русском если выбрать - "Russe", на французском, если выбрать - "Francais"
