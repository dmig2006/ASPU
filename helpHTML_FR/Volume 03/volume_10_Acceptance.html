Sous-systeme "Admission"

В конец документа

В подсистеме "Прием" предоставлена возможность пополнения элементов ЗИП в машине. Внешний вид окна представлен на рисунке 1.


Рисунок 1 - Подсистема прием

Назначение кнопок:

  1. - для перехода к последней надписи.
  2. - для перехода к предыдущей надписи.
  3. - для перехода к следующей надписи.
  4. - для перехода к первой надписи.
  5. - увеличение количества элементов, в случае если принятая информация не верна.
  6. - уменьшение количества элементов, в случае если принятая информация не верна.
  7. - удаление элемента из перечня принятых элементов.
  8. "Накладная" - вывод накладной по принятым элементам (Становится доступной после приема).
  9. "Инф.листок" - вывод отчета по элементам показанным в перечне принятых (Становится доступной после приема).
  10. "Сохранить" - запись принятых элементов в базу данных.
  11. "Помощь" - позволяет открыть справку.
  12. "Выход" - осуществляет выход из окна подсистемы "Прием".

Поля:

  1. "Наименование" - Имя элемента.
  2. "Обозначение" - Обозначение элемента.
  3. "Местоположение" - местоположение элемента в формате с.с.т.щщщ.яяя. Только цифры и '.', смотри пункт 3.5(Подсистема "ЗИП").
  4. "Количество" - Количество полученных элементов.

Для пополнения количества элементов в системе АСПУ необходимо выполнить следующие действия:

  1. Курсор(мигающая вертикальная линия) должен находиться в окне ввода, где будут отражаться считываемые терминалом штрих-коды, иначе коды элементов не будут занесены в список полученных элементов.
  2. Считывание терминалом:
    • Подвести терминал к штрих-коду.
    • Увидеть что элемент отобразился в поле наименование.
    Ввод с клавиатуры:
    • Ввести штрих код в поле.
    • Нажать на клавиатуре клавишу "Enter"
    • Увидеть что элемент отобразился в поле наименование.
  3. Когда все элементы будут зарегистрированы, необходимо нажать кнопку "Сохранить" и информация будет сохранена.После этого откроется окно смотри рисунок 2.


    Рисунок 2 - Экспедитор
  4. Становится доступной клавиша "Накладная" для вывода на печать информации о принятых элементах, экспедиторе, принимающем, дате и времени произведенной операции смотри рисунок 3.


    Рисунок 3 - Накладная
  5. Для вывода на печать перечня принятых элементов, нужно нажать клавишу "Инф.листок", смотри рисунок 4.


    Рисунок 4 - Инф.листок
  6. При необходимости повтора операции приема, необходимо выйти в главную форму, и войти в подсистему "Прием" повторно.
