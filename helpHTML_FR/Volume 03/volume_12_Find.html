В конец документа

В подсистеме "Поиск" предоставлена возможность поиска элементов ЗИП-Г по атрибутам. Внешний вид окна представлен на рисунке 1.


Рисунок 1 - Подсистема поиск

Для поиска элемента необходимо в соответствующие поля ввести атрибуты поиска.

Назначение полей:

  • "Комплект" - выбор комплекта искомого элемента из выпадающего списка, либо его ввод вручную. Любые символы.
  • "Обозначение" - ввод обозначения элемента. Только цифры.
  • "Наименование" - ввод наименования элемента. Любые символы.
  • "Штрих-код" - ввод штрих кода элемента, вводится с клавиатуры или с помощью терминала. Только целые числа.
  • "№ склада" - номер склада где хранится элемент. Только целые числа.
  • "№ стеллажа" - номер стеллажа где хранится элемент. Только целые числа.
  • "№ тумбочки" - номер тумбочки где хранится элемент. Только целые числа.
  • "№ ящика" - номер ящика где хранится элемент. Только целые числа.
  • "№ ячейки" - номер ячейки где хранится элемент. Только целые числа.

При поиске по местоположению элемента введите номера склада, стеллажа(шкафа), ящика(панели) и ячейки в соответствующие поля ввода. Для игнорирования какого-либо атрибута местоположения, необходимо оставить пусто в соответствующем поле.

При любом изменении данных в выше описанных полях, выполняется автоматический поиск элементов. Результат поиска отражается в списке. Для выделения искомого элемента из списка необходимо сделать его активным в таблице, щелкнув на нем левой кнопкой мыши, чтобы он был выделен цветом.

Если вход в подсистему "Поиск" был произведен из главной формы, то при нажатии кнопки "Об элементе", будет выполнен переход в подсистему "ЗИП" на выбранный элемент. Иначе, при нажатии кнопки "Переход.", произойдет переход в подсистему из которой была вызвана подсистема "Поиск", и активным станет найденный элемент.

Для очистки полей ввода атрибутов необходимо нажать кнопку "Очистить".

Для вызова справки используется кнопка - "Помощь".

Для выхода в главное меню, нажать кнопку - "Выход".
