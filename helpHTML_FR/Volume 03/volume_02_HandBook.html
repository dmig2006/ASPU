В подсистеме "Справочник" предоставлена возможность добавления, удаления и корректировки информации содержащей в главной базе данных. Внешний вид окна представлен на рисунке 1.


Рисунок 1 - Окно подсистемы справочник

Назначение кнопок:

  1. Кнопка "Поиск" позволяет, найти необходимый элемент для удаления или редактирования. При нажатии на этой кнопки станет активным окно поиска смотри п.3.12 (Подсистема "Поиск").
  2. Кнопка "Добавить" позволяет добавить новый элемент в базу данных. При нажатии этой кнопки открывает окно, представленное на рисунке 2.


    Рисунок 2 - Добавление элемента в БД

    Для добавления нового элемента в базу данных необходимо заполнить обязательные поля - "штрих-код" и "наименование". Если у склада с выбранным номером изменить наименование в текущем элементе, то наименование склада будет изменено для всех элементов ЗИП с данным номером склада. В случае, когда остальные поля незаполнены, выходит сообщение "Данные введены не полностью. Будете вводить?". Нажать кнопки "Да" - означает возврат в режим редактирования элемента, "Нет" - элемент сохранится в базе данных с незаполненными полями.
    • Поле "Штрих-код" - штриховой код элемента. Только целые числа.
    • Поле "Обозначение франц" - обозначение элемента на французском. Любые символы.
    • Поле "Обозначение рус" - обозначение элемента на русском. Любые символы.
    • Поле "Наименование франц" - полное наименование элемента на французском. Любые символы.
    • Поле "Наименование рус" - полное наименование элемента на русском. Любые символы.
    • Поле "Комплект франц" - имя комплекта, к которому принадлежит на французском. Любые символы.
    • Поле "Комплект рус" - имя комплекта, к которому принадлежит на русском. Любые символы.
    • Поле "Местоположение" - местоположение элемента в формате с.с.т.щщщ.яяя. Только цифры и '.', смотри пункт 3.5(Подсистема "ЗИП").
    • "№склада" - номер склада где хранится элемент. Только целые числа.
    • "№стеллажа" - номер стеллажа где хранится элемент. Только целые числа.
    • "№тумбочки" - номер тумбочки где хранится элемент. Только целые числа.
    • "№ящика" - номер ящика где хранится элемент. Только целые числа.
    • "№ячейки" - номер ячейки где хранится элемент. Только целые числа.
    • Поле "Наименование склада" - имя склада где хранится элемент.
    • Поле "Габариты(длина х ширина х высота)" - габариты элемента. Любые символы.
    • Поле "Единица измерения франц" - французская единица измерения элемента. Любые символы.
    • Поле "Единица измерения рус" - русская единица измерения элемента. Любые символы.
    • Поле "Количество (максимальное)" - максимальное количество элементов. Только цифры и '.'.
    • Поле "Количество (имеющееся)" - сколько всего элементов.Только цифры и '.'.
    • Поле "Дата изготовления (дд\мм\гггг)" - дата изготовления элемента. Только цифры.
    • Поле "Срок годности (в месяцах)" - гарантийный срок использования элемента в месяцах. Только цифры.
    • Поле "Примечание франц" - Дополнительная информация об элементе(в том числе информация об особых условиях хранения) на французском. Любые символы.
    • Поле "Примечание рус" - Дополнительная информация об элементе(в том числе информация об особых условиях хранения) на русском. Любые элементы.
    • Поле "Рисунок" - рисунок элемента. Рисунок формата *.bmp (* - означает любое имя)
  3. Кнопка "Изменить" позволяет редактировать выбранный элемент в базе данных. При нажатии на кнопку открывается окно редактирования представленное на рисунке 3.

  4. Кнопка "Удалить" позволяет удалить выбранный элемент из базы данных.
  5. Кнопка "Помощь" позволяет открыть справку.
  6. Кнопка "Выход" осуществляет выход из окна подсистемы "Справочник"
